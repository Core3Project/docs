# Analytics

Simple page view tracking with a dashboard widget. No external services.

| | |
|---|---|
| **Default** | Enabled |
| **Slug** | `analytics` |

## What It Does

Tracks page views by incrementing a view counter on each post. The view count is shown in the admin post listing and on the dashboard stats.

## Settings

This module has no configurable settings.

## Activation

Enable from **Admin → Modules**.
