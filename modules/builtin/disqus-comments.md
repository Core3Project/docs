# Disqus Comments

Replace the built-in comment system with Disqus. Disqus handles comment display, moderation, spam filtering, and notifications.

| | |
|---|---|
| **Default** | Disabled |
| **Slug** | `disqus-comments` |

## What It Does

Replaces the native Core 3 comment system with Disqus. Uses the `comments_replace` hook to inject the Disqus embed script instead of the built-in comment form and listing.

## Settings

| Setting | Description |
|---|---|
| `disqus_shortname` | Disqus Shortname. Your Disqus site shortname. Find it at https://disqus.com/admin → Settings → General. |

## Activation

Enable from **Admin → Modules**. After activation, configure the settings from the module's settings page.
