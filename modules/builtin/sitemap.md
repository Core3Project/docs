# Sitemap

Auto-generates /sitemap.xml for search engines.

| | |
|---|---|
| **Default** | Enabled |
| **Slug** | `sitemap` |

## What It Does

Registers a `/sitemap.xml` route that generates a standard XML sitemap listing all published posts, pages, and categories. Search engines can use this to discover and index your content.

## Settings

This module has no configurable settings.

## Activation

Enable from **Admin → Modules**.
