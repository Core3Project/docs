# Search

Adds site search with a search box widget and /search results page.

| | |
|---|---|
| **Default** | Enabled |
| **Slug** | `search` |

## What It Does

Adds a `/search` route with a search results page. Searches post titles and content using SQL `LIKE` queries. Integrates with the search widget for the sidebar.

## Settings

This module has no configurable settings.

## Activation

Enable from **Admin → Modules**.
