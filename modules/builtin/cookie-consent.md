# Cookie Consent

GDPR-compliant cookie consent banner. Dismissable with customizable message.

| | |
|---|---|
| **Default** | Disabled |
| **Slug** | `cookie-consent` |

## What It Does

Displays a dismissable cookie consent banner at the bottom of the page. The banner text and button label are configurable. Uses a cookie to remember the user's choice so the banner only appears once.

## Settings

| Setting | Description |
|---|---|
| `cookie_message` | Banner Message. The message displayed in the cookie banner. |
| `cookie_button` | Accept Button Text. Text for the accept/dismiss button. |
| `cookie_policy_url` | Privacy Policy URL. Optional link to your privacy policy page. |

## Activation

Enable from **Admin → Modules**. After activation, configure the settings from the module's settings page.
