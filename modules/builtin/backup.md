# Backup

One-click database backup. Download a full SQL dump from the admin panel.

| | |
|---|---|
| **Default** | Disabled |
| **Slug** | `backup` |

## What It Does

Adds a database backup download button to the admin panel. Generates a SQL dump of all Core 3 tables and sends it as a downloadable `.sql` file. Only accessible to admin users.

## Settings

This module has no configurable settings.

## Activation

Enable from **Admin → Modules**.
