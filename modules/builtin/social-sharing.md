# Social Sharing

Adds share buttons (X/Twitter, Facebook, LinkedIn, Email) to the bottom of posts.

| | |
|---|---|
| **Default** | Disabled |
| **Slug** | `social-sharing` |

## What It Does

Adds share buttons (Twitter/X, Facebook, LinkedIn) after each post's content. The buttons use simple share URLs — no external JavaScript is loaded.

## Settings

| Setting | Description |
|---|---|
| `share_networks` | Networks. Comma-separated: twitter, facebook, linkedin, email, reddit, whatsapp |

## Activation

Enable from **Admin → Modules**. After activation, configure the settings from the module's settings page.
