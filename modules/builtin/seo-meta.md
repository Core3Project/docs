# SEO Meta

Add custom meta titles, descriptions, and Open Graph tags to posts and pages for better SEO and social sharing.

| | |
|---|---|
| **Default** | Enabled |
| **Slug** | `seo-meta` |

## What It Does

Injects Open Graph meta tags and a meta description into the `<head>` of every page. Uses the post excerpt or a configurable site-wide meta description.

## Settings

| Setting | Description |
|---|---|
| `seo_title_suffix` | Title Suffix. Appended to every page title (e.g. ' — My Blog'). Leave empty for none. |
| `seo_default_image` | Default OG Image URL. Used when a post has no featured image. |

## Activation

Enable from **Admin → Modules**. After activation, configure the settings from the module's settings page.
