# Introduction

Core 3 CMS is a lightweight, open-source PHP blogging platform. The entire CMS weighs in at around 160 KB — roughly 400x smaller than a default WordPress installation.

It runs on virtually any PHP host without requiring Composer, Node, or command-line access. Upload the files, run the four-step installer, and you're publishing in under a minute.

## Key Concepts

**Posts & Pages** — Posts are blog entries with dates, categories, and comments. Pages are static content like "About" or "Contact".

**Themes** — Control the look of your site. Themes use PHP templates and CSS. Child themes can inherit from a parent theme, overriding only what they need.

**Modules** — Extend functionality without touching core code. Modules hook into named extension points to inject content, register routes, or modify data.

**Admin Panel** — A WordPress-style admin interface with sidebar navigation, a WYSIWYG editor, media management, user roles, and appearance customisation.

## Architecture

```
core3cms/
├── index.php              # Front controller
├── core/
│   ├── bootstrap.php      # Loads everything, defines C3_VERSION
│   ├── config.php         # Generated by installer (not in repo)
│   ├── config.sample.php  # Template with placeholders
│   ├── classes/           # Core PHP classes
│   └── functions/         # Helper functions
├── admin/
│   ├── index.php          # Admin router
│   ├── layout.php         # Admin shell (sidebar, topbar)
│   └── pages/             # Admin page controllers
├── content/
│   ├── themes/            # Installed themes
│   ├── modules/           # Installed modules
│   └── uploads/           # User uploads
├── assets/
│   ├── images/            # Logos, favicons
│   └── js/                # Editor JS
└── install/               # Installer (delete after setup)
```

## Requirements

| Requirement | Minimum |
|---|---|
| PHP | 7.4 or later |
| MySQL | 5.7+ or MariaDB 10.3+ |
| Web server | Apache with `mod_rewrite` |
| PHP extensions | `pdo_mysql`, `mbstring`, `json` |
